# Two main keys per thumb.

I find that my thumb naturally rests 1u from the index finger home position and I think that this is the perfect boundary
between the two main thumb keys!
For aesthetic reasons and since the space in the center of the keyboard is hard to reach anyway, I want to thumb keys
to be bigger than 1u. The size of the central keys needs to adapt, so that the boundary falls where I want it.
But the other two thumb keys can be 1.25 or 1.5u depending on how much space is left for other keys in the bottom row.
The idea is that key sizes there should be relatively uniform, 1.25u by default, possibly with the outermost one being
1.5u on account of little pinky curling into a ball to hit it.

## Key sizings depending on the gap size between hand positions.

With 3u between the index home keys, which affects only the Ergoslat:
 - the perfect alignment would require 1u keys; I have so far rejected it for aesthetic reasons.
 - next best would be 1.25u ==> TODO let's add this as an option.
 - 1.5u are not ergonomically optimal, but on the Ergoslat, they allow to use only two keycap sizes for everything,
   which is what that's the current option.

With 4u between the index home keys, this is naturally achieved by having 1.5u keys in the center,
so I'll do that on all such boards.

With 4.5u between the index home keys, it's done with 1.75u keys in the center.

With 5u between the indexes, there are many options:
 - 1.5u main thumb keys and a central (not super-easy to reach) 1u key to fill the gap. ==> done on Ergoplank 15/5
 - 1.75u main thumb keys and leaving a 0.5u gap. ==> maybe offer as option somewhere
 - big 2u thumb keys, which gives nice traditional vibes, but needs stabilizers and keycaps can't be exchanged with other places on the board, so that's a no.
 - 1.75u main thumb keys and missing the perfect spot ==> done on Ergoboard 16/5

With 5u between the indexes:
 - 1.75u main thumb keys and a central (not super-easy to reach) 1u key to fill the gap.
   ==> done on Ergoboard 16/5.5. Don't need more options here, since this is not my favorite board anyway.
